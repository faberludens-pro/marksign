# MarkSign

Convert any document to Markdown. On your Mac, in seconds.

**Status:** Alpha v1.0 · macOS · Free during testing

---

## Download

[**MarkSign 1.0 for macOS →**](https://github.com/faberludens-pro/marksign/releases/latest)

Requires macOS 13 or later. Apple Silicon and Intel supported.

---

## What it does

MarkSign converts PDF, DOCX, EPUB, PPTX, HTML, and image files to clean, structured Markdown — locally, with no internet connection required and no data sent anywhere.

- Drag and drop files onto the app
- Conversion runs locally using [docling](https://github.com/DS4SD/docling), PyMuPDF, and MarkItDown
- Preview the result instantly in the companion viewer
- Output saved next to the original file
