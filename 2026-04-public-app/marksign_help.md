# MarkSign Help

**MarkSign** converts documents to clean Markdown — locally, on your Mac, with no cloud upload.

---

## Getting started

1. **Drop files** onto the MarkSign window, or click **Select Files**
2. Choose a destination folder (defaults to Downloads)
3. Click **Convert**
4. When done, click **Preview** to read the result, or **Saved in →** to open the folder

---

## Supported formats

| Format | Extension | Notes |
|--------|-----------|-------|
| PDF | `.pdf` | Scanned PDFs require docling (first-run download) |
| Word | `.docx`, `.doc` | Full paragraph and heading support |
| PowerPoint | `.pptx`, `.ppt` | Slides converted to sections |
| EPUB | `.epub` | Book-quality markdown output |
| Excel | `.xlsx`, `.xls` | Tables preserved as Markdown tables |
| Plain text | `.txt` | Encoding detection included |
| Rich Text | `.rtf` | Inline formatting converted |

---

## Preview

Click **Preview** on any converted file to open it in **MarkSign Preview** — a dedicated Markdown viewer with the Newsprint reading theme. The preview auto-reloads if the file changes on disk.

---

## Menu bar icon

The MarkSign shield icon in your menu bar gives quick access to:
- **Convert file** — bring the window to front
- **About MarkSign**
- **Help**
- **Quit MarkSign**

---

## First-run setup

On first launch, MarkSign downloads **Docling** — an AI-powered converter for complex PDFs (scanned pages, multi-column layouts, tables). This is a ~1–2 GB one-time download. Simpler PDFs work immediately without it.

---

## Privacy

MarkSign processes all files **locally** on your Mac. No files are uploaded to any server. No internet connection is required after the first-run Docling download.
