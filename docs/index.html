What it does

Your documents, in the format that matters

PDF, Word, ePub, PowerPoint, Excel, and RTF — MarkSign converts them all to clean, structured Markdown using a local pipeline built on docling, PyMuPDF, and MarkItDown. No cloud. No subscription. No copy-paste. The converted file appears next to the original, ready to use.

How it works

Fits into how you already work

🖱

Right-click to convert

Select one or more files in Finder, right-click, and choose Convert with MarkSign. Batch conversion with no window required.

⬇︎

Drag and drop

Drop any file onto the MarkSign window and the Markdown file appears next to the original — named, dated, and ready to use.

→ Batch conversion — multiple files at once

Always in your menu bar MarkSign runs quietly in the Mac menu bar — no Dock icon, no distractions. One click opens the converter or brings up your recent conversions, from wherever you are.
