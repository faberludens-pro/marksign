Getting started

Three steps

Open the DMG — double-click the downloaded file.

Drag MarkSign to Applications — just like any Mac app.

Open MarkSign — it will install the conversion engine on first launch (takes a minute, one time only).

Private by design. MarkSign runs entirely offline. Your documents are never uploaded anywhere. Conversion happens on your Mac using open-source tools — docling, PyMuPDF, and MarkItDown.
